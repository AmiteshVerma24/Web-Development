FlexBox properties

1. flex-direction

Defines how flex items are aligned

(i) row

(ii) column

2. flex-wrap

Defines how the elements of flex container aligns

(i) no-wrap

(ii) wrap

3. Flex flow

It is combination of both flex direction and flex wrap

eg: flex-flow: row wrap

4. Justify content

It aligns the content within the flex container in main axis

(i) flex-start

(ii) flex-end

(iii) center

(iv) space-around

(v) space-between

5. Align Items

It aligns the content within the flex container in cross axis

(i) flex-start

(ii) flex-end

(iii) center

(iv) stretch

(v) baseline
